### 🚀 Featured Projects

#### 🦯 [Smart Blind Stick](https://github.com/prithiv16official-tech/smart-blind-stick)
An IoT-based assistive device for visually impaired users, built during my internship at **RGreen Technologies**.
- Obstacle detection using **ESP32 + HC-SR04** ultrasonic sensors
- Real-time **GPS location tracking** with **GSM** module for emergency alerts
- Live data pipeline: ESP32 → **Supabase (PostgreSQL)** → web dashboard
- Web dashboard with login system for caregivers to monitor location in real time
- 🔧 In progress: battery monitoring, gyroscope-based fall detection, water sensor, voice assistance
